### git命令集
	git init ---------------------- 将项目加入git管理
	git add . --------------------- 将项目下的所有文件、文件夹放入到git
	git status -------------------- 查看文件变化
	git commit -m <...> ----------- 提交信息
	git remote add origin --------- 连接到github仓库
	git pull ---------------------- 远程获取最新版本至本地
	git push -u origin master ----- 最后将提交的本地文件上传到远程（第一次会要求输入账号）

	git add -u -------------------- 修改/删除的文件 
	git add -A -------------------- 提交所有改动（包括新增）
	git add <path/filename> ------- 指定文件
